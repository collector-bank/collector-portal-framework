# Spinner

This component should be shown when retrieving data or performing slow computations.

## Props

| Name        | Required | Type    | Possible values                               |
|-------------|----------|---------|-----------------------------------------------|
| `size`      |          | number  |                                               |
| `centered`  |          | boolean |                                               |

## Example

```jsx
<Spinner />
```
