# ActionButton
This is a button inteded to either trigger actions or features, or link to a place where you can do them

## Props
| Name    | Required | Type          | Possible values |
| ------- | -------- | ------------- | --------------- |
| `icon`  |          | `JSX.Element` | 16px svg icon   |
| `label` |          | `string`      |                 |
